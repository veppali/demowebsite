Welcome to the main page.

Explore the website to see different "webapps". The whole website is (excluding the game demo) was made Django framework, python, css and javascript.

Blog

In the blog you can create a user account and start creating your own blog posts as well as edit them after the fact. The search bar adds ability to search the blogs with tags and keywords. You can also comment on other people's blog posts. (Email authentication and personal page are planned).

Translator

The actual translation functionality is by libretranslate. The translator uses the libretranslate api to get translations to the users input text.

Game

The game demo is a sidescrolling space themed shoot em up game, made with an open source Godot Engine and exported into html5 webassembly game. You can examine the source code of the game at the link provided.
